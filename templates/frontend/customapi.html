Welcome to ApiHub, a platform where you can discover and integrate various APIs for your projects. (This Feature is For Our subscribers Only).
Create Your Custom Api's, define your own structure of api as well as define validations on the object fields.
Define you endpoint, and all the endpoints are unique.

Create API (POST, GET)

Success : 201 Error : 400, 415, 422

Create Your Custom Api, required fields are name of you api, endpoint of your api, validations contain your class fields and validations on them, and a public flag to make api private or public(default). 

{
                "name" : "Age Checker",
                "endpoint" : "agecheck",
                "success_response" : {"message" : "Valid age!"},
                "error_response" : {"message" : "Invalid age!"},
                "validations":{
                        "age" : {"type" : "int", "min" : 18, "max" : 65},
                        "name" : {"type" : "str"}
                    },
                "public" : true
                }

Post Data and Get Data from your end points (POST, GET)

Success : 201, 200 Error : 400, 404, 415, 422

Now you can Use your api, post data on api, get data through api. Each data validated through your custom validations. We provide a lot of validations to include in your fields.
Your url for api should be '/api/endpoint/{Your End Point Name}/' 

{
                "age" : 20,
                "name" : "Ashish",
                }

Fields and Validations

Create your Own Custom Fields and their type of validations
To create fields write your fields inside validation parameter and their validation in the a dictionary or object format, remeber to declare the type of field also
Data Types of fields are integer, decimal, string, boolean
Types of Validations : For integer and decimal -> type, min, max Types of Validations : For string -> type, min_length, max_length, regex 

{"validations" : {"height" : {"type" : "integer", "min" : 165, "max" : 175},
                             "password" : {"type" : "string", "min_length" : 8, "max_length" : 20, "regex" : "^(?=.*[A-Za-z])(?=.*\d)(?=.*[@$!%*#?&])[A-Za-z\d@$!%*#?&]{8,}$" },
                             "indian" : {"type" : "boolean"}
                            }
    }
